## Key Questions
1. 三个元素的上下间距如何做到像素级等距一致？
   - 答：将头像、名字输入框、系别 Badge 放到同一个 Flex 容器中，设定统一的 `gap`，并移除输入框本身的不对称 padding (如 py-0.5)，使视觉间隙完全相等。
2. 当不是“有现蛋”时如何填补卡片空白？
   - 答：设计精致的提示小底色条（如“正在孵”、“已撤窝”、“投资”对应的不同主题条），高度与“现蛋数量”输入框一致，起到视觉平衡的作用。
3. 长图导出时，新加的提示条和标题是否需要过滤？
   - 答：长图需要完美保留这些兜底提示条，因为它们体现了当前的真实状态；但要确认没有按钮元素，因此不会被 action-buttons 类过滤掉。

## Decisions Made
- [决策]: 使用 `gap-1.5` 作为头像、精灵名、系别的统一间距。
- [决策]: 统一主标题样式：`text-lg font-bold text-slate-800`，说明字体：`text-xs text-slate-500 font-medium`，图标大小统一为 `w-5 h-5`。
- [决策]: 非“有现蛋”状态在下方以精致的小条带（带呼吸灯或前置圆点）展示状态说明，保持与“有现蛋”时高度一致。
